透明性レポート

AI Code Builder は、ユーザーのプライバシーとデータセキュリティを第一に考えて設計されています。以下にデータの取り扱いについての詳細を報告いたします。

1. データの保存と利用

  • ローカルファースト: プロジェクトのソースコードやチャット履歴は、すべてお使いのブラウザ内部（IndexedDB）に保存されます。当方のサーバーに保存されることはありません。
  • AI学習の非利用: 入力されたプロンプトや生成されたコードは、LLM（大規模言語モデル）の学習データとして使用されることはありません。

2. APIキーの保護

  • 暗号化: アプリ内で設定されたAPIキーは、Web Crypto API (AES-GCM) を用いてブラウザのローカルストレージ内で強力に暗号化されます。
  • 通信の安全性: APIキーは、コード生成のために指定されたLLMプロバイダー（OpenAI, Groq等）の公式エンドポイントへのみ直接送信され、中継サーバー等を経由しません。

3. 外部通信について

  • 本アプリケーションは、ユーザーの利便性を高めるため、以下の外部サービスと通信を行います：
    • LLM API: ユーザーが設定したプロバイダーのAPI（コード生成用）
    • CDN: Monaco Editor、フォント等のアセット読み込み
  • 上記以外のトラッキング、分析（アナリティクス）、または第三者へのデータ提供を行うための通信は一切ありません。

4. セキュリティ対策

  • 生成されたコードの中には、危険な操作（システムの破壊、意図しない外部通信等）が含まれる可能性があるため、アプリ内には簡易的な危険コード検知（Danger Detection）システムを導入し、ユーザーに警告を表示する仕組みを設けています。
